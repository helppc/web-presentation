# HelpPC — landing page

Jednostránkový rozcestník organizace **HelpPC** (webové aplikace a knihovny pro vývojáře).
Statická stránka, dvojjazyčná (CZ / EN), respektuje logomanuál HelpPC.

Single-page hub for the **HelpPC** organization. Static, bilingual (CS / EN), follows the HelpPC brand manual.

## Obsah / Contents

- `index.html` — celá stránka (inline CSS + JS, přepínač jazyka, `localStorage`)
- `assets/` — loga (SVG), favicony, OG obrázek z brand balíčku

## Spuštění / Run

Otevři `index.html` v prohlížeči, nebo nasaď složku na libovolný statický hosting
(GitHub Pages, Netlify, Cloudflare Pages, …). Žádný build.

Open `index.html` in a browser, or deploy the folder to any static host — no build step.

## Brand

Barvy a typografie dle HelpPC logomanuálu: HelpPC Blue `#1769FF`, Deep Black `#080B10`,
Dark Canvas `#030713`, písmo Inter / system-ui, diagonální motiv „vpřed".
Loga se používají pouze z dodaných souborů (beze změn barev, deformací a efektů).
